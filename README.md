# dulce-tentacion
Sitio web estático de la pastelería Dulce Tentación desarrollado con HTML
📖 Descripción del proyecto

Este proyecto consiste en el desarrollo de un sitio web estático utilizando HTML para la pastelería ficticia “Dulce Tentación”. El objetivo es presentar de manera ordenada la información del negocio, incluyendo su presentación, productos, servicios y datos de contacto.

La página web ha sido diseñada con una estructura sencilla y clara, utilizando etiquetas básicas y semánticas de HTML5, lo que permite una navegación fácil y una mejor organización del contenido.

🍩 Contenido del sitio web

El sitio incluye las siguientes secciones:

🏠 Inicio
🧁 Productos
📍 Nosotros
📞 Contacto
💻 Tecnologías utilizadas
HTML5
🌐 Enlace del sitio web

👉 https://guille2007jesus-lang.github.io/dulce-tentacion/

🎯 Objetivo del proyecto

Aplicar los conocimientos básicos de desarrollo web, utilizando HTML para crear una página estática funcional que represente un negocio real.

📚 Conclusión

Este proyecto permite comprender la estructura básica de una página web, el uso de HTML y la importancia de organizar correctamente la información para mejorar la experiencia del usuario.
